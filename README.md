# IFRS_parser

## Описание
* на основе данных macrotrends.net выгружаются данные по финансовой отчётности компаний
* производится чистка выбросов в данных – необходимо провести исследование разных методик корректировки выбросов
* генерятся метрики по темпам роста, уровню закредитованности, производительности
* считаются мультипликаторы
* создаются таблицы и графики для сравнения нескольких компаний
